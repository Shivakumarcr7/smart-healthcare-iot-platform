# 🏥 Smart Healthcare IoT System using ESP32 & FreeRTOS

An enterprise-grade Smart Healthcare IoT System built using **ESP32** and **FreeRTOS** that demonstrates real-time patient monitoring, concurrent embedded programming, IoT communication, and fault-tolerant system design.

The project consists of **six major modules**, showcasing advanced FreeRTOS concepts such as multitasking, queues, semaphores, mutexes, watchdog timers, adaptive scheduling, and reliable IoT communication.

---

## 📌 Project Overview

This project simulates a smart healthcare monitoring system capable of monitoring patient vitals, controlling medical devices remotely, adapting monitoring frequency intelligently, and ensuring reliable operation even during network failures.

The system is designed to resemble an enterprise IoT healthcare solution suitable for hospitals, telemedicine, and remote patient monitoring.
